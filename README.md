# ASE2026_Vibe_Slicer

> By: Thomas Martin Pfister  
> Advanced Systems Engineering S2026, University of Salzburg  

An attempt to create a Slicer for a 3D Printer using only AI Tools. The program will take a 3D Object in STL format as input, and output usable GCODE for a FDM 3D printer.

Current Stage: Conversion of STL file to usable intermediate JSON data in layers for further use.
